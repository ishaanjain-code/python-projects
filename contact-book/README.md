
# Contact Book (Python)

A simple **console-based Contact Book application** written in Python.  
It allows users to add, delete, and search contacts using a menu-driven interface.

---

## Features
- Add a contact with name and 10-digit phone number
- Prevent duplicate names and numbers
- Delete contacts using **name or phone number**
- Search contacts by name
- Simple and interactive CLI menu

---

## Concepts Used
- Dictionaries
- Functions
- Loops
- Conditional statements
- Input validation
- String methods
