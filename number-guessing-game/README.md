# Number Guessing Game (Python)

A simple **console-based number guessing game** built using Python.  
The program generates a random number within a user-defined range and guides the user to guess it with hints.

---

## Features
- User-defined number range
- Random number generation
- Hints for higher or lower guesses
- Attempt tracking
- Replay option after each game

---

## Concepts Used
- Loops (`while`)
- Conditional statements
- Exception handling (`try / except`)
- Random number generation
- User input handling
- Basic statistics (mean)
