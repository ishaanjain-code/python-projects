
# Password Strength Checker (Python)

A simple **console-based Password Strength Checker** built using Python.  
It evaluates a password based on multiple criteria and provides feedback on its strength.  
The program can also generate strong password suggestions when required.

---

## Features
- Checks password strength (Weak / Medium / Strong)
- Validates password based on:
  - Minimum length
  - Digits
  - Uppercase letters
  - Lowercase letters
  - Special characters
- Generates strong password suggestions
- Interactive user input

---

## Strength Criteria
A password is evaluated using the following conditions:
- Minimum length of 8 characters
- Contains at least one digit
- Contains uppercase and lowercase letters
- Contains at least one special character

Strength is determined based on how many conditions are satisfied.
