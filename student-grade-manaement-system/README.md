
# Student Grade Management System (Python)

A simple **console-based Student Grade Management System** built using Python.  
It collects student details, subjects, and marks, then calculates percentages.

---

## Features
- Input multiple students
- Input multiple subjects per student
- Store data using nested dictionaries
- Calculate total percentage for each student
- Display results on the console

---

## Concepts Used
- Functions
- Dictionaries (nested)
- Loops
- User input handling
- Arithmetic operations
