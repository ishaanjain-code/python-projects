# To-Do List App (Python)

A simple **console-based To-Do List application** written in Python.  
It allows users to add, view, and delete tasks using a menu-driven interface.

---

## Features
- Add new tasks
- View all tasks
- Delete tasks by task number
- Simple and interactive CLI menu

---

## Concepts Used
- Lists
- Functions
- Loops
- Conditional statements
- Exception handling
- User input handling
